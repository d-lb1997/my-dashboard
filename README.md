# my-dashboard

Scaffolded Vite + Vue 3 + TypeScript project prepared for `pnpm`.

Quick start (run these in the project root):

```bash
# install dependencies with pnpm
pnpm install

# run development server
pnpm run dev
```

Notes:
- This repo is prepared to use `vuetify`, `@mdi/font`, `chart.js`, and `vue-chartjs`.
- If you don't have `pnpm` installed, install it with one of the following:

```bash
# via corepack (recommended on modern Node versions):
sudo corepack enable && sudo corepack prepare pnpm@latest --activate

# or via npm:
sudo npm install -g pnpm
```
